# 智能菜谱助手 (Smart Recipe Helper)

一款基于微信小程序的智能菜谱推荐应用，帮助用户发现、学习和收藏美味佳肴。

## 项目特点

- 🍳 **丰富的菜谱库** - 包含荤菜、素菜、主食、汤类等多种分类
- 🔍 **智能搜索** - 支持按菜名、食材、描述等多维度搜索
- ⭐ **收藏功能** - 收藏喜欢的菜谱，随时查看
- 📱 **精美界面** - 现代化UI设计，流畅的用户体验
- 📤 **社交分享** - 一键分享给好友或朋友圈

## 技术栈

- **框架**: 微信小程序原生开发
- **语言**: JavaScript (ES6+)
- **样式**: WXSS
- **数据存储**: 本地缓存 (wx.setStorageSync)

## 项目结构

```
smart-recipe-helper/
├── app.js                 # 小程序入口文件
```

## 功能模块

### 1. 首页 (index)
- 今日推荐菜谱（随机展示热门菜谱）
- 分类快捷入口
- 热门菜谱横向滚动列表
- 搜索功能入口

### 2. 发现页 (discover)
- 菜谱搜索（支持关键词搜索）
- 分类筛选（全部/荤菜/素菜/主食/汤类）
- 排序功能（默认/点赞数/时间/难度）
- 菜谱列表展示

### 3. 详情页 (detail)
- 菜谱完整信息展示
- 食材清单（支持复制）
- 详细制作步骤
- 收藏/取消收藏功能
- 分享功能（好友/朋友圈）

### 4. 收藏页 (favorite)
- 已收藏菜谱列表
- 删除单个收藏
- 清空所有收藏
- 快捷跳转发现页

### 5. 个人中心 (profile)
- 用户信息展示与登录
- 统计数据（收藏数/菜谱数/分类数）
- 常用功能菜单
- 关于应用信息
- 缓存清理功能

## 开发规范

### Git 提交规范

本项目使用规范的 Git 提交格式：

```
feat: 新功能
fix: 修复问题
docs: 文档更新
style: 代码格式调整
refactor: 重构代码
test: 测试相关
chore: 构建/工具相关
```

### 代码规范

1. **命名规范**
   - 变量和函数：camelCase
   - 常量：UPPER_SNAKE_CASE
   - 类：PascalCase
   - 文件名：kebab-case 或 camelCase

2. **注释要求**
   - 关键业务逻辑必须添加注释
   - 复杂算法需要详细说明
   - 函数需添加功能说明

3. **样式规范**
   - 使用 rpx 单位适配不同屏幕
   - 遵循 BEM 命名规范
   - 保持样式复用性

## 开发环境要求

- 微信开发者工具 v1.05+
- Node.js v14+ (如需使用 npm)
- 基础的微信小程序开发知识

## 运行步骤

1. 克隆或下载本项目
2. 使用微信开发者工具打开项目目录
3. 在 `project.config.json` 中填入你的 AppID（或使用测试号）
4. 点击编译按钮运行项目
5. 在模拟器中预览效果

## 数据说明

项目中的菜谱数据存储在 `data/recipes.js` 文件中，包含：
- 12 道精选菜谱
- 完整的食材清单
- 详细的制作步骤
- 分类和难度标签

后续可扩展为从服务器获取数据。
